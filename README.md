## 📱 About The Project
Eventify is a modern Android application built with **Jetpack Compose** that helps users track upcoming events with visual countdown timers. Features a bold neo-brutalism design aesthetic.

## ✨ Features (Phase 1 - Prototype)

### Current (Day 1)
- ✅ Modern Jetpack Compose UI
- ✅ Material Design 3 components
- ✅ Interactive state management
- ✅ Click counter demo
- ✅ Toast feedback system

### Planned (Upcoming)
- 🎨 Neo-brutalism color palette
- 📋 Event list with RecyclerView (LazyColumn)
- ➕ Add event screen
- 🔍 Event details screen
- 🎯 Event filtering (Academic, Personal, Occasion)
- ⏱️ Static countdown display
- 🎭 Smooth animations

## 🚀 How to Run This Project

### Prerequisites
- Android Studio Hedgehog (2023.1.1) or newer
- JDK 17 or higher
- Android SDK with API 24+
- Git installed

### Steps
1. **Clone the repository:**
```bash
   git clone https://github.com/YOUR_USERNAME/eventify-android.git
```

2. **Open in Android Studio:**
    - File → Open
    - Navigate to the cloned folder
    - Click OK

3. **Wait for Gradle sync** (first time takes 3-5 minutes)

4. **Run the app:**
    - Click green Play button (▶️)
    - Select emulator or connected device
    - Wait for build and deployment
